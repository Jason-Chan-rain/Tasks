### [功能描述]：
根据数学公式计算计算机存储容量的需求、存储器的价格以及所需成本  
计算机存储容量公式：M = 4080*e的0.28(Y-1960)次方  
存储器的价格：P1 = 0.3*0.72的(Y-1974)次方 （美元/位）  
若计算机字长为16位，则 P2 = 0.048*0.72的(Y-1974)次方 （美元/字）  
上述公式中 Y 代表年份，M 是存储容量（字数），P1和 P2代表价格
